2. Для чего используется псевдокласс :hover?
	изменение ссылка, на которой находится указатель
3. Для чего предназначен псевдокласс :active?
	изменение  ссылка в момент нажатия.
4. Для чего предназначен псевдокласс :link?
	изменение непосещенная ссылка
5. Для чего предназначен псевдокласс :visited?
	изменение посещенная ссылка
6. Как создать вертикальную панель навигации?
	Создать список со ссылками
	Удалить маркеры списка.(list-style-type: none;)
	Изменить ссылку до блочного элемента, так как элемент <a> является строчным 
	элементом(display:block)
7. Как создать горизонтальную панель навигации?
	 Создать маркированный спискок: 
	 Преобразовать пункты списка в строчные элементы display: inline; 
	 Для ссылок установить display: inline-block 
8. Как создать панель навигации с выпадающими элементами?
	.nav>ul>li {position: relative;display: block;}
	.nav li .second {display: none; position: absolute;}
	.nav li:hover .second{display:block}
	<nav class="nav"><ul><li><ul class="second"><li></li></ul></li></ul>
8. Для чего используется метод CSS-спрайт?
Для смены одной графической ссылки на другую при наведении на нее указателя мыши. 
Использует единственное изображение для представления различных состояний одной
навигационной кнопки
9. Что представляет из себя ролловер?
 смена одного изображения другим
10. Каким образом скрыть выпадающие элементы?
	с помощью псевдокласса hover и смены свойства display с none на block
	.nav li .second {display: none; position: absolute;}
	.nav li:hover .second{display:block}
11. Для чего используется свойство position?
	Свойство position определяет тип позиционирования, используемого для элемента из 
	пяти разных значений
12. Какие типы позиционирования Вы знаете?
static (статический) — означает, что расположение элементов соответствует их расположению в 
HTML-коде;
relative (относительный) — позиционируется относительно своего исходного положения;
fixed (фиксированный) — позиция элемента блокируется в определенной позиции на экране;
absolute (абсолютный) — позволяет поместить элемент в любой позиции страницы с точностью до 
одного пиксела относительно ближайшего позиционированного предка или окна просмотра;
sticky (закреплённый) — позиционируется на основе позиции прокрутки пользователя.
